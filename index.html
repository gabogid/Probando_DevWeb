¿Qué es CSS?

Las hojas de estilo en cascada (CSS) se utilizan para formatear el diseño de una página web. Con CSS, puede controlar el color, la fuente, el tamaño del texto, el espacio entre los elementos, cómo se colocan y distribuyen los elementos, qué imágenes de fondo o colores de fondo se van a usar, diferentes pantallas para diferentes dispositivos y tamaños de pantalla, ¡y mucho más!

CSS se puede agregar a documentos HTML de 3 maneras:

  • En línea

    • mediante el uso del atributo de estilo dentro de los elementos HTML

    Un encabezado azul

    Un párrafo rojo.

  • Interno

    • Se utiliza un CSS interno para definir un estilo para una sola página HTML.

  • Externo

    • Se utiliza una hoja de estilo externa para definir el estilo de muchas páginas HTML. Para usar una hoja de estilo externa, agréguele un enlace en la sección HEAD de cada página HTML:
